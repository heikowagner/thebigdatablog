- [Cuda Capable Nvidia GPU](https://www.amazon.de/s/ref=nb_sb_noss?__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&url=node%3D430181031&field-keywords=nvidia&rh=n%3A340843031%2Cn%3A427956031%2Cn%3A430161031%2Cn%3A430181031%2Ck%3Anvidia&tag=addonsdeaddonssh) (>GeForce 8)

- [Cuda Toolkit](https://developer.nvidia.com/cuda-downloads)

Adding GPU Support in Matlab is very easy. In case of the Code given above the only necessary change needed is to replace *line 45* by *Z = gpuArray.zeros( T, pp );*and to call the function with Arrays X,Y,x,… which has been loaded to the GPU Memory using *gpuArray() ,* to return the data to the workspace one can then simply call *gather().\*

To compare the performance between GPU and CPU a simple simulation with \_\_WPMATH\_0000\_\_ is carried out. In detail we simulate a mixture of normal distributions given by\
\begin{equation}\
X(t\_j)= \epsilon\_j\
\end{equation}\
observed at a uniform random grid with \_\_WPMATH\_0001\_\_ observations \_\_WPMATH\_0002\_\_.
