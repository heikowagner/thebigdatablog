Suppose multivariate data given by a pair ![(y,x)](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-57b6e6ee680857c53c52b227b4b67ad1_l3.png "Rendered by QuickLaTeX.com") where the explanatory variable ![x \in \mathbb{R}^p](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-6baf5012ca650182746df3586c28b0cc_l3.png "Rendered by QuickLaTeX.com") and ![y \in \{1, \dots, c+1\}](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-dbf1a2a372f7dcae4aa6338b9e6715bd_l3.png "Rendered by QuickLaTeX.com"). Instead of prediction ![y](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-0af556714940c351c933bba8cf840796_l3.png "Rendered by QuickLaTeX.com") based on ![x](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-ede05c264bba0eda080918aaa09c4658_l3.png "Rendered by QuickLaTeX.com") within one stage, like for example using ordinary least squares, we model ![y](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-0af556714940c351c933bba8cf840796_l3.png "Rendered by QuickLaTeX.com") using ![M](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-10ebb71bad275c1815a8f2a8c5dea0be_l3.png "Rendered by QuickLaTeX.com") hidden stages (layer). The first layer will then be modeled using ![x](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-ede05c264bba0eda080918aaa09c4658_l3.png "Rendered by QuickLaTeX.com") with a certain activation function ![g()](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-12578d36d6cdc13f400105a2b7206200_l3.png "Rendered by QuickLaTeX.com") the second layer will be modeled using the first layer and so on. This is very similar to a multi-stage regression model to predict the group affiliation ![y](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-0af556714940c351c933bba8cf840796_l3.png "Rendered by QuickLaTeX.com"). A very special cases of a neuronal network are with ![c=1](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-8bc0f94c08cf058d316895e763d34082_l3.png "Rendered by QuickLaTeX.com"), ![g()=\frac{1}{e}](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-8bc5a67d97a1f9601fb02d500301463e_l3.png "Rendered by QuickLaTeX.com") and one-stage is for example the logistic regression or with ![c \geq 1](https://www.thebigdatablog.com/wp-content/ql-cache/quicklatex.com-6f9f79b2108b89d14be0636d5bb66662_l3.png "Rendered by QuickLaTeX.com") a multi nominal regression. Using c=p … we end up with a simple linear regression.
