### Abstract

In this Article we will show how to setup a use Kubernetes and Docker to setup a scalable Cassandra cluster. ~~(Since we are dealing with~~ ~~timeseries~~ ~~we will usev~~ ~~KariosDB~~~~)~~. At each Cassandra Node we intend to attach a Spark Node to perform computation on the data stored in the Cassandra Database. The Database will be filled by several Node.js server using the Twitter rest API to collect tweets in parallel. To analyze the Data we will in addition add a Zeppelin Notebook to the main spark Node.
